# Skills

A personal collection of Claude skills built for real work — content, strategy, product thinking, and brand. These are the skills I use across my own projects and client work.

## What’s in here

| Skill | What it does |
|-------|-------------|
| `brand-pillars.skill` | Distills brand strategy into a set of core, actionable pillars |
| `content-strategy.skill` | Plans content using the 60/20/20 framework: authority, personal, community |
| `extract-main-ideas.skill` | Pulls the key ideas from an article before repurposing it for social |
| `linkedin-emotional-triggers.skill` | Diagnoses why a LinkedIn post isn’t resonating and adds emotional depth |
| `product-thinking-diagrams.skill` | Produces lightweight pre-design artifacts: entity models, IAs, page blueprints, and more |

## How to use

These are [Claude Skills](https://docs.claude.com) — instruction files that tell Claude how to handle specific tasks in a repeatable, consistent way.

To use a skill, add it to your Claude Code project’s `.claude/skills/` folder, or reference it directly in a conversation.

## Notes

These skills are personal and opinionated. They reflect my voice, my frameworks, and how I like to work. Fork and adapt freely.
